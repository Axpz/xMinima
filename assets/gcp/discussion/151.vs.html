App Engine Standard vs Flexible 对比

特性 App Engine Standard App Engine Flexible
运行环境 沙盒环境（受限） 基于 VM，自定义容器
语言支持 支持特定语言（如 Python、Node.js） 几乎所有语言，自定义容器
启动速度 快速启动 启动较慢
扩缩容速度 快速自动 稍慢但更灵活
是否可访问 VPC ❌ 需配置 Serverless VPC Access ✅ 默认可访问
是否支持 SSH ❌ 不支持 ✅ 支持
是否支持访问本地数据库 需要 VPC Connector + VPN 原生支持 VPN，直接访问
适合场景 轻量 Web 应用，高自动化 复杂业务，自定义需求

📡 架构路径对比

🟦 App Engine Standard → 本地数据库

  App Engine Standard（受限环境）
           │
    Serverless VPC Access Connector
           │
        VPC 网络（Private Subnet）
           │
      Cloud VPN / Cloud Interconnect
           │
         本地数据库（On-Prem）

🟩 App Engine Flexible → 本地数据库

  App Engine Flexible 实例（直接在 VPC 中）
           │
        VPC 网络（Private Subnet）
           │
      Cloud VPN / Cloud Interconnect
           │
         本地数据库（On-Prem）
