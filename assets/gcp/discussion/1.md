> **metoxig（4周零1天前）**    
> ✅ 答案：D. 在负载均衡器后面为每个 API 路径使用单独的后端池。    
> **理由：**    
> - 保持现有 SSL/DNS 单一入口点    
> - 支持路径路由，引导流量到相应版本    
> - 客户端无需更改    
> - 各版本清晰分离    
> - 常见 API 版本控制模式    
> 👍 1
